This repository contains implementation details and optimized solutions for various data structures and algorithms. My focus has been on building a deep intuition for complex logic rather than just memorizing syntax, specifically targeting high-tier software engineering placements.

## 🛠️ Tech Stack
* **Language:** C++ 
* **Environment:** Git/GitHub for version control 
* **Logic Focus:** Time & Space Complexity Analysis ($O(n)$, $O(\log n)$, etc.)

## 📂 Repository Structure
The repository is organized by topic to ensure easy navigation:

* **Linear Data Structures**
    * `Arrays/` - Searching, sorting, and manipulation logic.
    * `Linked-Lists/` - Singly, Doubly, and Circular implementations.
    * `Stacks-Queues/` - Applications in recursion and scheduling.
* **Non-Linear Data Structures**
    * `Trees/` - Binary Trees, BST, and Heap implementations.
    * `Graphs/` - Traversal (BFS/DFS) and Shortest Path algorithms.
* **Advanced Topics**
    * `Recursion-Backtracking/` - Solving complex decision-based problems.
    * `Dynamic-Programming/` - Optimization for overlapping subproblems.
